# AASHA Connect (Frontend Prototype)

AASHA Connect is an offline-first web application designed for AASHA workers, admins, and families.

## Key Features
- Offline data collection using local storage / IndexedDB
- Family registration with QR generation
- Admin verification dashboard
- Family QR scan → view details + govt schemes
- Mock Aadhaar Autofill

## Folder Structure
- /worker → Worker pages
- /admin → Admin pages
- /family → Family QR pages
- /assets → Images
- index.html → Landing page
